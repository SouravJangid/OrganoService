
# OrganoService

## ----------Command for Getting Project Locally----------

### Clone the repository
```bash
git clone https://github.com/SouravJangid/OrganoService
```

### Navigate into the project directory
```bash
cd OrganoService
```

### Backend Setup
```bash
cd backend
npm install
npm run server
```



###Open in new terminal for client



### Frontend Setup
```bash
cd ../client
npm install
npm start
```
